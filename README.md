# cantinhodogueimer

Clube de jogos do Discord dos Biladeiros. Site estático em
[biladeirosgit.github.io/cantinhodogueimer](https://biladeirosgit.github.io/cantinhodogueimer/).

Os dados são todos manuais e vivem em `src/cdg/gameData.json`.

## Scripts

- `npm start` — servidor de desenvolvimento
- `npm run build` — build de produção
- `npm run deploy` — publica no GitHub Pages (branch `gh-pages`)
